# seeyon_saveExcel
致远OA_saveExcel任意文件上传exp (2023/06版本修复) 

# Usage
```
Usage：go run seeyon_saveExcel.go -u http://xx.xx.xx.xx -f xx.jsp (-c)
```

```
Options:
  -c string
        cookie [Optional]
        
  -f string
        file [Required]
        
  -u string
        target url [Required]
```
